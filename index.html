Root cause fixed The underlying technical issue has been identified and resolved. A recurrence would require a new failure — not the same code path. Verified from Jira releases or explicit confirmation in task comments.
Mitigated Customer-facing impact has been stopped or significantly reduced, but the full systemic fix is still in progress or tracked in a dedicated follow-up task or initiative.
